THIS SUBROUTINE 1. UPDATES THE DIFFERENCE TABLE FROM THE PREVIOUS STEP (IF NOT DONE ALREADY). 2. PREDICTS WHAT THE VALUES OF THE DEPENDENT VARIABLES, Y, AND THE DIFFERENCE TABLE, DT, WILL BE AT THE END OF THE CURRENT STEP.

Y = VECTOR OF PREDICTED VALUES COMPUTED BY THIS SUBROUTINE. YN= VECTOR OF VALUES OF Y COMPUTED ON THE LAST STEP. F = VECTOR OF DERIVATIVE VALUES. DT= ARRAY CONTAINING DIFFERENCE TABLES. KD= VECTOR GIVING ORDERS OF THE DIFFERENTIAL EQUATIONS (IF EQUATIONS HAVE DIFFERENT ORDERS). KQ= VECTOR OF INTEGRATION ORDERS.
